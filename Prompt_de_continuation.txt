## RÉSUMÉ FONCTIONNEL PAR MODULE

### config.py (v2.3.5)
Constantes centrales immutables (MappingProxyType/frozenset/tuple).

- **WEIGHTS** (11 critères, Σ=1.0) : geology 0.20, tree_species 0.16,
  canopy_openness 0.14, altitude 0.09, slope 0.09, twi 0.08,
  dist_water 0.06, terrain_roughness 0.06, aspect 0.04,
  ground_cover 0.04, disturbance 0.04
- **Altitude** : optimal (200,600), range (150,900),
  bonus gaussien centré 350m
- **Pente** : seuils non chevauchants 10/20/35/45°
- **TWI** : optimal (6.0,10.0), dry<3→floor 0.10, wet>14→floor 0.10,
  waterlog>18→éliminatoire
- **Essences** : 22 espèces scorées, unknown=0.25,
  châtaignier=0.80 (PAS éliminatoire). ELIMINATORY_SPECIES set.
- **Géologie** : 17 catégories, granite/gneiss/siliceux éliminatoires.
  _GEOLOGY_KEYWORD_MAP : 32 patterns BDCharm-50.
  resolve_geology() : cascade 3 niveaux (direct→normalisé→keywords).
  1086/1086 résolus.
- **Floors** : LANDCOVER_FOREST_FLOOR=0.80, DIST_WATER_FOREST_FLOOR=0.15
- 7 basemaps Géoplateforme + OSM + Esri + OpenTopo

### data_loader.py (v2.3.3)
Chargement données brutes. WFS IGN/BRGM + fallback local/synthétique.

- load_dem() : lecture fenêtrée (src.index + Window positionnels)
- _normalize_geology() : DESCR prioritaire, cascade 3 étapes vectorisée,
  _BDCHARM_DESCR_OVERRIDES (21 patterns). 1232/1232 = 100%.
- Normalise colonnes (essence_canonical, geology_canonical, etc.)

### grid_builder.py (v2.3.5)
Grille régulière L93, calcul de 11 scores [0,1].

- **Terrain** : compute_terrain() → altitude, slope, aspect,
  roughness, TWI. _compute_twi() = D8 vectorisé + tri topologique
  (boucle Python ~4M pixels, ~10-30s estimé, NON MESURÉ).
- **Scores terrain** : altitude (bonus gaussien), slope (seuils),
  roughness (σ local), aspect (sinusoïdal 170°), twi (zone optimale)
- **Scores écologiques** : dist_water (EDT + bonus type),
  tree_species (rasterize max), geology (rasterize max).
  Stockent rasters int-codés + lookups pour hotspot enrichment.
- **Micro-habitat** : canopy (transition 30m), ground_cover
  (forêt-only), disturbance (lisières + urbain)
- **Masques** : urban (buffers), water (all scores→0),
  landcover (forest floor + green_clip × _VEGETATION_CRITERIA)
- **_VEGETATION_CRITERIA** = (canopy, ground_cover, disturbance)
  — tree_species EXCLU (fix #37)
- **Attributs exposés** : scores dict, eliminatory masks,
  _raw_tree_species, _landcover_green_score, _twi,
  rasters int-codés + lookups, score_confidence

## DÉCISIONS D'ARCHITECTURE VERROUILLÉES ⚠️

Ne PAS proposer d'alternative à ces choix sans demande explicite :

| # | Décision | Justification |
|---|----------|--------------|
| D1 | TWI algorithme D8 (pas D∞/MFD) | Suffisant pour 25m, complexité moindre |
| D2 | DESCR prioritaire sur NOTATION pour géologie | NOTATION donnait 14.7% non résolus vs 0% avec DESCR |
| D3 | Châtaignier score 0.80, PAS éliminatoire | Association fréquente morilles post-perturbation |
| D4 | tree_species hors _VEGETATION_CRITERIA | Évite double pénalisation par green_score (fix #37) |
| D5 | apply_urban_mask AVANT micro-habitats | Sinon disturbance recalculé sur zones masquées |
| D6 | Rasters int-codés + lookups pour hotspots | Performance vs string matching sur clusters |
| D7 | Landcover cache-only (pas de téléchargement) | Stabilité, reproductibilité |
| D8 | Forest floor 0.80 pour cellules BD Forêt | Évite sous-notation forêts par landcover HSV |
| D9 | dist_water floor 0.15 en forêt | Cours d'eau temporaires non cartographiés |
| D10 | Pénalité couverture NaN-safe floor=0.5 | Évite score=0 si quelques critères manquants |

**Approches REJETÉES :**
| Proposition | Raison du rejet |
|---|---|
| score_slope seuils renforcés (0,8)/(15)/(25)/(45) | Supersédé par TWI |
| dist_water pénalité engorgement <5m | Supersédé par TWI waterlog |

## CONVENTIONS PYLANCE (P1-P10)

P1. `from __future__ import annotations` en tête
P2. Types modernes : `dict` pas `Dict`, `str | None` pas `Optional`
P3. Attributs Optional → variable locale + assert + type hint
P4. `getattr()` + `isinstance()` pour attributs optionnels grid
P5. `np.asarray()` autour de rasterize/gaussian/EDT/zoom/.values/.isin.
    `np.any()/np.sum()/np.all()` au lieu de méthodes Series
P6. Import direct si sous-module échoue
P7. `# type: ignore[…]` pour : Window(...), .html.add_child(...),
    _read_vector_file, _normalize_geology, _generate_synthetic_geology,
    load_hydro, load_urban
P8. Scores dict : isinstance guard avant .shape
P9. Logger `logging.getLogger("cartomorilles.<module>")` — jamais print
P10. Immutabilité : MappingProxyType, frozenset, tuple sur constantes
